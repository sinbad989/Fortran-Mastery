# Control Structures

- courses of action in an algorithm requires the following: 
  - logical expression and logical operators 
  - one or more blocks of statemets
- `if then endif` 
- `if then else if endif` 
- `case` statement 
- `do ... enddo` loop
  - `do` loop
  - `do while` form 
  - `do ... if then exit enddo` 
  - `cycle`  statement
  - `exit` statement
